# Unregelmäßige Verben

## Hilfsverben

| Infinitiv | Präteritum | Partizip II | Präsens      |
| --------- | ---------- | ----------- | ------------ |
| haben     | hatte      | gehabt      | unregelmäßig |
| sein      | war        | gewesen     | unregelmäßig |
| werden    | wurde      | geworden    | unregelmäßig |
